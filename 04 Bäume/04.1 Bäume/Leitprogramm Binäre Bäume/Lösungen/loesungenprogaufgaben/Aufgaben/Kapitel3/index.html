Binärbäume

Programmieraufgabe 3.2

Schreiben Sie eine Methode preorderPrint() in der Klasse BinaryNode. Diese Methode soll alle Knoten eines Binärbaumes in der Preorder-Reihenfolge ausgeben.

Die Klasse BinaryNode ist bereits vorgegeben. Entsprechende Stellen im Code die ergänzt werden müssen sind mit Kommentaren markiert. Führen Sie die Klasse Aufgabe3_2 aus, um ihre Implementierung zu überprüfen.
